## 5. Issues Encountered

1. **Bugcrowd direct scraping**: bugcrowd.com/programs is a SPA with heavily obfuscated output. Could not extract meaningful program list from direct URL. bbradar.io was essential as a structural proxy.
2. **bbradar.io rate limiting**: Pages 3+ for Bugcrowd and some deeper pages timed out. The aggregator shows 207 Bugcrowd programs but we could only view ~75 of them in detail.
3. **VDP vs Bounty distinction**: Intigriti shows "Hide VDP programs (51)" meaning many programs are unpaid VDPs. Bbradar.io hides them by default but we need to verify each one manually.
4. **Managed Bug Bounty duplicates**: Bugcrowd's MBB migration means some programs appear twice (old listing + new MBB listing). Need deduplication logic.
